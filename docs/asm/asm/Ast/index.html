  1. | Mov : [< `Reg_64_imm_a ] mnemonic
  2. | Push : [< `Reg_64 ] mnemonic

  4. | Xor : [< `Reg_64_reg_64 ] mnemonic
  5. | Syscall : [< `Nothing ] mnemonic
  6. | Pop : [< `Reg_64 ] mnemonic
  7. | Jmp : [< `Label ] mnemonic
  8. | Sub : [< `Reg_64_imm_a ] mnemonic
  9. | Cmp : [< `Reg_64_imm_a ] mnemonic
  10. | Je : [< `Label ] mnemonic
  11. | Jne : [< `Label ] mnemonic
  12. | Jle : [< `Label ] mnemonic
  13. | Movq : [< `Reg_64_reg_128 | `Reg_128_reg_64 ] mnemonic
  14. | Punpckhqdq : [< `Reg_128_reg_128 ] mnemonic
  15. | Movapd : [< `Reg_128_reg_128 ] mnemonic
    (*

    Real instruction name, not the cat jumped on the keyboard

    *)
  16. | Pinsrq : [< `Reg_128_reg_64_imm_a ] mnemonic
  17. | Addpd : [< `Reg_128_reg_128 ] mnemonic
  18. | Mulpd : [< `Reg_128_reg_128 ] mnemonic
  19. | Haddpd : [< `Reg_128_reg_128 ] mnemonic
    (*

    Not sure that these instructions can really be used with integer values in xmm registers. The specification of the instructions is a complete mess

    *)

Some (actually most of them) mnemonic variants are not supported, but it's easy to add them later
